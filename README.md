# FusionIME
FusionIME packs various input methods for CJKV languages into one app.

## Features
* Multi-language typing without switching between input methods. Unified themes and styles.
* Rich support for CJKV languages. (Mozc, Google Pinyin, Zhuyin, Hanja, Han Nom)
* Language Tab-Bar for easy switching between multiple languages.
* Support for various keyboard layouts. (Colemak, Dayi, JIS, 3-Set Korean, ...) Suggestions for new layouts are welcome!

## Supported Languages and Layouts
### Latin Languages
* Languages: English, German, Spanish, French, Italian, Brazilian Portuguese
* Layouts: QWERTY, Dvorak, Colemak

### Chinese
* Simplified & Traditional Chinese Pinyin
* Traditional Chinese Zhuyin
* Chinese Cangjie, Quick, Dayi

### Japanese
* Romaji QWERTY, Kana JIS, Kana Syllables

### Korean
* Hangul 2-Set
* Hangul 3-Set 390, 3-Set Final
* Old Hangul 2-Set, 3-Set 393

Supports Hanja conversion for both modern and middle Korean.

### Vietnamese
* QWERTY, Telex

Supports Han Nom conversion.

## Dictionary Data
* Korean: [Korean Standard Dictionary](https://stdict.korean.go.kr), [libhangul](https://github.com/libhangul/libhangul/tree/main)
* Vietnamese: [chunom.org](https://chunom.org/)
