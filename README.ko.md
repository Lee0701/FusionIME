# FusionIME
FusionIME는 한중일월(한자문화권) 언어를 위한 여러가지 입력 방식을 단일 앱으로 제공합니다.

## 기능
* 앱 간 전환 없이 다국어로 입력할 수 있습니다. 통일된 스타일과 테마로 지원하는 모든 언어를 사용할 수 있습니다.
* 한중일월 언어를 지원합니다. (Mozc, 구글 병음, 주음, 한국어 한자, 쯔놈)
* 여러가지 언어 간의 전환을 빠르게 할 수 있는 탭바를 제공합니다.
* 다양한 키보드 배열을 제공합니다. (콜맥, 대이수입법, 가나 JIS, 세벌식 등) 새로운 배열이 필요하다면 제안해 주세요!

## 지원하는 언어와 배열
### 라틴 계열 언어
* 언어: 영어, 독일어, 스페인어, 프랑스어, 이탈리아어, 브라질 포르투갈어
* 배열: QWERTY, 드보락, 콜맥

### 중국어
* 간체·번체 중국어 한어병음
* 번체 중국어 주음부호
* 창힐수입법, 속성수입법, 대이수입법

### 일본어
* 로마자 QWERTY, 가나 JIS, 가나 오십음도

### 한국어
* 한국어 두벌식 표준
* 한국어 세벌식 390, 세벌식 최종
* 옛한글 두벌식, 세벌식 393

단어·어절단위 한자 변환과 정음 한자 변환을 지원합니다.

### 월남어
* QWERTY, 텔렉스

한놈 변환을 지원합니다.
